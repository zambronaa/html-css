Principais formatações

Negrito / Destaque

Podemos usar a tag <b> como negrito Não semantica

Ou a tag <strong> como destaque Semantica

Italico / Ênfase

Nesta frase, temos a tag <i> em italico Não semantica

Nesta frase, para colocar em ênfase usamos a tag<em> Semantica

Texto Marcado

Para marcar um texto usamos a tag <mark> para marcar um texto

Texto grande e pequeno

Com o grade podemos usa o <big> mas logo não irá funcinar no html5

Para escrever texto pequeno usamos a tag <small> para ficar pequiqniniho

Texto excluido e Texto inserido

Se quiser excluir um texto ASSIM DESSE JEITO podemos usa a tag >del<

Com o inserido podemos usar a tag >ins< para INSERIR O TEXTO e é SEMANTICÁ

E também podemos usar a tag >u< PARA FICAR DESSE JEITO nÃO SEMANTICÁ

Texto sobrescrito e subscrito

TEXTO SOBRESCRITO usamos a tag <sup> por exemplo X10+3

TEXTO SUBSCRITO podemos usar a tag <sub> H2O
